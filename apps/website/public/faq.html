Frequently asked questions

What is Marketer MCP?

Marketer MCP is a public hub of AI tools built specifically for marketing operations. It uses the Model Context Protocol (MCP) standard so your AI assistant can discover and call marketing tools — budget pacing, campaign QA, channel allocation — through a single endpoint.

Who is it for?

Marketing leaders, ops teams, and agencies who manage budgets across paid search, paid social, display, video, and organic channels. If you spend time in spreadsheets reconciling spend or checking campaign setups, Marketer MCP automates that work.

What is Budget Buddy?

Budget Buddy is the first live tool on Marketer MCP. It takes your monthly budget and channel-level spend as JSON input and returns a pacing report with over-budget alerts. Response time is under 200ms.

How do I connect?

You configure your AI assistant (Claude, GPT, or any MCP-compatible client) to connect to the Marketer MCP endpoint. Tool execution requires a Bearer token. Discovery can be public or restricted by your environment policy. See the quickstart for details.

Is my data secure?

Yes. All tool execution requires token authentication. Requests are rate-limited, payloads are validated against schemas, and tool execution is namespaced. Structured logs with request IDs provide full auditability. See the security page for details.

What does it cost?

Marketer MCP is free during the beta period. Pricing details for the general availability launch will be published before the beta ends. There is no per-seat model — you pay for usage, not team size.

Does it work for agencies with multiple clients?

Yes. Namespaced tool execution means each client's data stays isolated. One connection serves your entire book of business. Auth tokens can be scoped per team or per client.

What tools are on the roadmap?

After Budget Buddy: Channel Allocation Planner, Campaign Flighting Helper, Attribution QA, Creative QA, and Audience Planner. Each tool follows the same pattern — structured input, structured output, built for the specific workflow it serves.

How do I get early access?

Join the launch list and we will send you an API token and onboarding guide when your spot opens.
