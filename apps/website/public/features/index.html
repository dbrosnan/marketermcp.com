First live tool

Budget Buddy

Budget Buddy is our first live tool — a marketer-native assistant for budget planning, spend tracking, and channel allocation. Send your budget data and campaign spend; get rollups, alerts, and pacing insights in structured JSON.

  • Budget vs actual rollups — Channel-level and campaign-level summaries
  • Over-budget alerts — Instant visibility when spend exceeds plan
  • Channel allocation analysis — Compare planned vs actual mix
  • JSON input/output — Easy to automate and integrate
  • Real-time pacing — See where you stand against monthly targets

API usage

POST /v1/tools/budget_buddy.plan_overview/execute

Use Bearer auth. Send budget and spend data in the request body; receive structured rollups and alerts in the response.

How MCP works

Marketer MCP uses the MCP protocol with a single endpoint. Your AI assistant connects once, discovers available tools, and executes them with namespaced routes.

Connect

Point your AI assistant to the Marketer MCP endpoint. Authenticate with Bearer token.

Discover tools

Query the tool registry to list available tools, schemas, and parameters.

Execute

Call tools by name. Send structured input, receive structured output — no custom integrations.

Integration

Marketer MCP follows the MCP protocol — a standard for AI tool discovery and execution. Connect your AI assistant (Claude, Cursor, or any MCP-compatible client) to our hub endpoint.

  • Bearer auth — Secure execution with API tokens
  • Namespaced execution — Tools are called by namespace (e.g. budget_buddy.plan_overview)
  • Structured I/O — JSON schemas for inputs and outputs
