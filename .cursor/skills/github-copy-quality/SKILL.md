# GitHub-Inspired Copy Quality Skill

## Use when

- Writing homepage copy, docs copy, launch emails, or social posts.
- Reviewing messaging quality before publish.

## Outcome

Produces concise, outcome-first copy that follows the Marketer MCP voice and avoids hype.

## Steps

1. Read `docs/copy-system.md` and `docs/brand-guide.md`.
2. Enforce message flow:
   - pain
   - promise
   - proof
   - CTA
3. Check each section for:
   - one core idea
   - concrete language
   - no unsupported claims
4. Generate two variants:
   - direct/functional
   - premium/aspirational
5. Return final copy and a short rationale for chosen version.
