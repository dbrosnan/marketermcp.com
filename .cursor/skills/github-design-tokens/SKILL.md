# GitHub-Inspired Design Tokens Skill

## Use when

- Defining or updating brand colors, spacing, radius, and typography.
- Refactoring UI styles for consistency.

## Outcome

Moves one-off styles into reusable design tokens and keeps visuals consistent across pages.

## Steps

1. Read `docs/brand-guide.md`.
2. Extract repeated style primitives into token names.
3. Replace duplicated values in CSS with tokens.
4. Check for consistency across:
   - cards
   - buttons
   - headings
   - spacing rhythm
5. Suggest future automation path using Style Dictionary-compatible token shapes.
